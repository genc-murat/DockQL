Docker
Observability Language

A domain-specific language and CLI tool for querying, observing, and analyzing your Docker infrastructure in real time.

View on GitHub → Quick Start
dol 
# List running containers with high CPU
$ dol "observe containers | where cpu > 80% | sort cpu desc | select name, image, cpu"

# Stream container crash events
$ dol "events containers where action = die | select time, container, image"

# View container logs
$ dol "logs container my-app tail 50"

# Check Docker daemon connectivity
$ dol "ping"

Quick Start

Clone, build, and run your first query in under a minute.

terminal 
# Clone and build
$ git clone https://github.com/genc-murat/DockQL.git
$ cd DockQL
$ cargo build --release

# Run a query
$ ./target/release/dol "observe containers"

# Start the interactive REPL
$ ./target/release/dol repl

# Live-updating container monitor
$ ./target/release/dol top
cargo install dol --git https://github.com/genc-murat/DockQL
